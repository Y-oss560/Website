### 获取指定实例配置文件内容

地址

```
/api/protected_instance/process_config/file?uuid={{new_instance_uuid}}&remote_uuid={{service_uuid}}&fileName=spigot.yml&type=yml&apikey={{apikey}}
```

请求方式

```
GET
Content-Type: x-www-form-urlencoded; charset=utf-8
```
