### 创建实例

地址

```
/api/instance/?remote_uuid={{service_uuid}}&apikey={{apikey}}
```

请求方式

```
POST 
Content-Type: x-www-form-urlencoded
```

请求参数

```
command:cmd.exe	// 启动命令
cwd:.			// 工作目录
stopCommand:stop	// 停止命令
nickname:昵称{{$randomFileName}} // 实例昵称
ie:GBK	// 输入编码
io:GBK  // 输出编码

```
